== Synopsis
A hand-built raytracer, following the book "The Ray Tracer Challenge".

== Used libraries
* Cucumber-rs: https://github.com/cucumber-rs/cucumber

== License 
Apache (see LICENSE)

== Conventions
* Vectors and points have the format (x, y, z, w)
* w indicates whether it's a point (w = 1) or vector (w = 0)



== Running

=== Run tests
* cargo test
This will run all Cucumber feature tests
